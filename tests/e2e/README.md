# E2E Tests for Udemy Translation Extension

## Setup

1. Build the extension first:
   ```bash
   npm run build
   ```

2. Run the tests:
   ```bash
   npm test
   ```

## Test Coverage

- Extension loading verification
- Popup UI functionality
- Settings persistence
- Translation engine switching
- API key management

## Note

These tests require the extension to be built in the `dist` directory before running.
