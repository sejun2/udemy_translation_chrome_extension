# Privacy Policy for Udemy Subtitle Translator

**Last Updated: November 23, 2024**

## Overview

Udemy Subtitle Translator is a Chrome extension that translates Udemy video subtitles in real-time. This privacy policy explains how we handle your data.

## Data Collection and Storage

### What We Collect
- **Translation Settings**: Your preferences (enabled/disabled status, target language, original text display options)
- **DeepSeek API Key**: If you choose to use DeepSeek translation, your API key is stored locally

### Where Data is Stored
- All data is stored **locally on your device** using Chrome's `chrome.storage.sync` API
- Your DeepSeek API key never leaves your browser except when sent directly to DeepSeek's servers for translation requests
- We do **NOT** collect, store, or transmit any of your data to our servers

## Data Usage

### Translation Services
- **Chrome Built-in Translation**: No external data transmission
- **DeepSeek API**: When enabled, subtitle text is sent to DeepSeek's API servers for translation using your API key

### What We Do NOT Collect
- Personal information (name, email, etc.)
- Browsing history
- Course content or video data
- Any data beyond the translation settings mentioned above

## Third-Party Services

When using DeepSeek translation:
- Subtitle text is sent to DeepSeek API servers
- Please refer to [DeepSeek's Privacy Policy](https://www.deepseek.com/privacy-policy) for how they handle data

## Data Security

- Your API key is stored securely in Chrome's local storage
- All communications with DeepSeek API use HTTPS encryption
- No data is shared with third parties beyond the translation service you choose

## User Rights

You can:
- Delete all stored data by uninstalling the extension
- Clear your API key at any time through the extension popup
- Disable the extension without losing your settings
